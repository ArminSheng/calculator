# calculator
基于 Electron + javascript 实现的桌面计算器应用

## 运行
```
npm install
npm start
```

## 构建
```
npm run build:win
```
则会在项目中生成个 /计算器-win32-x64 文件夹，打开里面的 计算器.exe 即可打开计算器。
